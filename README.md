#Teknik Servis Ve Envanter Takip Sistemi
-----------------------------------------------------------------------------------------------------------------------------
Problem: Geleneksel yöntemlerde (kağıt/Excel) yaşanan veri kaybı, karmaşık stok takibi ve müşteri geçmişine ulaşma zorluğu.

Çözüm: Tüm servis süreçlerini merkezi bir veritabanında toplayan, kullanıcı dostu ve hızlı bir masaüstü yazılımı.

Hedef: Teknik servislerin operasyonel hızını artırmak ve hatasız finansal takip sağlamak.

Geliştirme Ortamı: Visual Studio 2022

Programlama Dili: C# (.NET Framework/Core)

Veritabanı: Microsoft SQL Server (MSSQL)

Veri Erişimi: ADO.NET (System.Data.SqlClient)

Tasarım: WinForms UI Controls
